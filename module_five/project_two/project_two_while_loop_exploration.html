Project Two: While-loop Exploration

Project Three ❯

In this project, we’ll learn about another kind of loop called a while-loop! We’ll also learn how to print text using a turtle! You’ll then get a chance to practice using while-loops and printing text.

For this project, you’ll need to create two turtles: one turtle to draw a line, and another turtle to do some running. The line will be drawn at a random x-coordinate, and it will represent a finish line! The runner turtle will then need to run towards the line until they cross it. Once the runner turtle crosses the finish line, they should say, “I win!”
